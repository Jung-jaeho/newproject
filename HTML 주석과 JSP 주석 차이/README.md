
# HTML 주석과 JSP 주석의 차이


## JSP에서 사용할수 있는 주석 
<br>

1. JSP주석 
~~~
    1. <%-- 주석내용 --%>
~~~

2. 자바 언어 주석<br>
~~~
     // 주석내용<br>
     /* 주석내용 */
~~~

3. HTML 주석
~~~
     <!-- 주석 내용 -->
~~~
